# KateStudio

Набор файлов проекта. В репозитории есть скрипт для индексации файлов.

Как использовать индексатор:

```bash
python3 scripts/index_repo.py
```

Это создаст `repo_index.json` в корне репозитория (список файлов, размеры, sha256 и метки времени).

Примечания:
- `repo_index.json` добавлен в `.gitignore` по умолчанию.
- Скрипт использует стандартную библиотеку Python (без внешних зависимостей).
