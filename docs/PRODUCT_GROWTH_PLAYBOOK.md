# Продукт и рост: лучшие практики + адаптация под K Sebe

## 1) North Star и продуктовая рамка

**North Star Metric (NSM)**: _количество завершённых практик в неделю на
активного пользователя_  
Почему: напрямую связано с ценностью (привычка) и предсказывает
удержание/монетизацию.

Вспомогательные метрики:

- Activation: первый “успешный опыт” в первые 24 часа
  (практика/дыхание/AI‑совет)
- Retention D7/D30
- AI cost/user/week (контроль экономики)
- Conversion Free → Premium (если вводим подписку)

## 2) Что делают успешные продукты (паттерны, не «цифры»)

Опорные референсы (как классы решений): `Down Dog`, `Alo Moves`, `Headspace`,
`Calm`, `Insight Timer`, `Mindbody`, `ClassPass`.

### 2.1 Персонализация с первого экрана

- onboarding‑quiz (цель/уровень/время/ограничения)
- “Практика дня” (1 тап)
- адаптивная длина (10/20/45 минут)

### 2.2 Habit‑loop (привычка)

- streak‑календарь
- “мягкие” пуши (не агрессивные), привязанные к цели пользователя
- weekly recap (“что поменялось в теле/настроении”)

### 2.3 Контент‑операции (без выгорания создателя)

- контент‑матрица: 70% evergreen, 20% сезонное, 10% эксперимент
- повторное использование: один урок → короткие клипы → пост → email → “практика
  дня”

### 2.4 Коммьюнити без токсичности

- челленджи без лидербордов “по всем” (лучше “я‑против‑себя”)
- шэринг в сторис (простая открытка с достижением)
- “вопрос Кате” (прямая связь усиливает доверие)

## 3) Лучшие решения для развития экосистемы (под ваш контекст)

### 3.1 K Sebe как “AI‑Inside Flow companion”

Уникальность: Inside Flow + русский язык + личный бренд + AI‑помощник.

Практичные фичи:

- **AI‑разбор** (фото/короткое видео) + “что делать завтра”
- **персональные микропрограммы** на 7 дней
- **дыхание/медитации** как ежедневный “вход”

### 3.2 Монетизация без ломки опыта

Подход: freemium‑воронка:

- Free: базовые практики + ограниченные AI‑сессии
- Premium: полная видеотека + программы + офлайн
- VIP: консультации/разборы + приоритет

## 4) “А что если?” (сценарии и проверяемые гипотезы)

### Сценарий A: AI‑First Studio

**Что если** Aria станет “ежедневным компасом”: состояние → рекомендация →
практика → рефлексия.  
Проверка: рост D7 retention и числа завершённых практик.

### Сценарий B: Inside Flow RU‑хаб

**Что если** K Sebe станет русскоязычным центром Inside Flow (контент, события,
партнёрства).  
Проверка: рост органики/рефералов, увеличение LTV.

### Сценарий C: “Studio OS”

**Что если** приложение станет операционной системой студии: расписание,
платежи, абонементы, CRM.  
Проверка: снижение ручной работы и рост повторных покупок.

## 5) Бэклог экспериментов (низкая стоимость → высокая ценность)

- [ ] Onboarding‑quiz (3–5 вопросов) + сохранение профиля целей
- [ ] “Практика дня” (простая эвристика → позже AI)
- [ ] Weekly recap (в приложении + пуш)
- [ ] Streak (без публичных сравнений)
- [ ] Шэринг “карточек” в сторис
